# 🎲 Habit Dice App

습관 형성을 게임처럼 재미있게 만들어주는 앱입니다. <br />
사용자는 매일 주어진 기회를 활용해 랜덤 보상을 얻고, 이를 통해 꾸준한 습관을 유지할 수 있습니다.
<br />
<br />

# 📌 Overview

이 앱은 단순한 체크리스트 기반의 습관 관리에서 벗어나,<br />
랜덤 보상 시스템(주사위)을 통해 사용자에게 동기부여를 제공합니다.

- 하루 1회 주사위 실행 제한
- 습관 수행 → 주사위 기회 획득
- 랜덤 보상/패널티 구조
- 게임 요소 기반 지속성 강화
<br />
<br />

# 🚀 Features

- ✅ 습관 생성 및 관리
- 🎯 하루 단위 습관 체크
- 🎲 주사위 시스템 (1일 1회 제한)
- 🎁 랜덤 보상 시스템
- 📊 습관 진행 상태 추적
<br />
<br />

# 🧠 Core Concept

>“습관을 게임처럼 만든다”

기존 습관 앱은 반복과 기록에 집중하지만,
이 앱은 불확실성과 기대감을 추가하여 사용자의 참여도를 높입니다.

- 행동 → 보상 구조 강화
- 랜덤 요소 → 도파민 유도
- 제한된 기회 → 희소성 확보
<br />
<br />

# 📖 Usage Scenario

1. 사용자가 습관을 생성한다.
2. 하루 동안 습관을 수행한다.
3. 수행 완료 시 주사위 기회를 획득한다.
4. 하루 1회 주사위를 굴린다.
5. 결과에 따라 보상 또는 효과를 받는다.
<br />
<br />

# 🎯 Goal

1. 습관 형성의 지속성 증가
2. 사용자 참여도 극대화
3. 게임화(Gamification)를 통한 자기관리 개선
